# Tic-Tac-Toe
A web-based Tic-Tac-Toe game built using HTML, CSS, and JavaScript, featuring dynamic gameplay logic, win/draw detection, and an interactive user interface.

## Features
- Two-player gameplay  
- Turn indication and win/draw detection  
- Highlighted winning combination  
- Restart option without page reload  
- Responsive design  

## Tech Stack
HTML, CSS, JavaScript

## How to Run
Clone the repository and open `index.html` in a web browser.
